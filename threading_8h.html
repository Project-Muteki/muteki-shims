◆ OSEnterCriticalSection()

void OSEnterCriticalSection ( critical_section_t *  mutex)

Enter/aquire a critical section (recursive mutex) context.

This will block when multiple threads are trying to enter the same context, but it will let repeated entries initiated by the same thread to pass through. The context is released when all of the entries are reverted by a OSLeaveCriticalSection() call.

Parameters
mutexThe critical section/mutex context.

◆ OSGetThreadPriority()

short OSGetThreadPriority ( thread_t *  thr)

Get the thread slot number. On Besta RTOS there is no explicit way to specify thread priority. Priority is implied in the natural order of the threads in the global thread table. Some slots in the table seem to be reserved (8 for the top and 18 for the bottom) and are not accessible by just allocating the thread with OSCreateThread(). User can move threads to these reserved slots by calling the OSSetThreadPriority() function.

Parameters
thrThe thread descriptor.
Returns
The slot number of the thread.
