int ExecuteProgram ( void *  prog,
int  invoke_subroutine,
const void *  app_arg1,
const void *  app_arg2 
)

Invoke a specific subroutine of the loaded app with arguments.

Once called, the control is fully transfered to the loaded app and this function will block until the invoked subroutine fully exits.

Actual format of arguments seem to be app-specific.

Parameters
progThe loaded executable description returned by LoadProgramA or LoadProgramW.
invoke_subroutineSubroutine to invoke.
app_arg1Argument 1. Can either be a pointer to some data or an integer.
app_arg2Argument 2. Can either be a pointer to some data or an integer.
Returns
The return value of the subroutine
